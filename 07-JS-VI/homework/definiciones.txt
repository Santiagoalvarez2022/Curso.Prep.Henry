* Funciones Callback

Funciones Callback : todas las funciones o "mini maquinas" reciben elementos (argumentos) para trabajar,
cuando se pasan funciones como argumentos se le llaman Callback y estas pueden usarse dentro de esta nueva 
funcion sin necesidad de volver a declararse.
